Points, Lines, and Planes Review:

  • Point: undefined, location only
  • Line: undefined, extends infinitely, straight
  • Plane: undefined, flat surface, infinite
  • Collinear Points: on the same line
  • Coplanar Points: on the same plane
Example:
  • Points A, B, C collinear → on same line
  • Points A, B, C, D coplanar → on same plane
