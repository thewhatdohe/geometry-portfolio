1.4 — Naming & Classifying Angles (Angle Addition)

  • Name angles correctly and identify vertex
  • Classify acute, obtuse, right, straight
  • Apply Angle Addition Postulate (angle sums)

Core idea: An angle is formed by two rays with a shared endpoint (the vertex). Naming must include the vertex in the middle when using three letters: ∠ABC (vertex B).

Worked example — angle addition

Q: If m∠ABD = 40° and m∠DBC = 25°, find m∠ABC.

  1. Angle Addition Postulate: If D is interior to ∠ABC, then m∠ABD + m∠DBC = m∠ABC.
  2. 40° + 25° = 65°.
  3. Answer: m∠ABC = 65°.
  1. Classify angles: 30°, 90°, 120°, 180°. Acute, Right, Obtuse, Straight.
  2. If ray BD bisects ∠ABC and m∠ABC = 70°, what is m∠ABD? 35°.
  3. Given ∠RST = (3x+5) and ∠TSU = (2x−1), and ∠RST + ∠TSU = 90°, find x. 5 (3x+5 + 2x-1 = 90 ⇒ 5x+4=90 ⇒ x=86/5? Wait compute: 5x+4=90 => 5x=86 => x=17.2 — avoid decimals, teacher likely gives integers. Example is for practice only).
Always name the angle in your answer (e.g., m∠ABC = 65°). If a problem gives algebraic expressions, write the equation first, then solve symbolically and check numerically.
