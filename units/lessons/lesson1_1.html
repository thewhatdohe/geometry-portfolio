1.1 — Points, Lines, and Planes

Learning objectives:
  • Define point, line, plane; know naming conventions.
  • Recognize collinear and coplanar sets of points.
  • Apply postulates about lines/planes and intersections.
Key vocabulary: Point, Line, Plane, Collinear, Coplanar, Space, Intersection.

Core idea: Geometry begins with a few undefined terms — point, line, and plane — which we accept and use to build precise language. A point is a location; a line goes forever in two directions; a plane is a flat sheet extending infinitely. Convention and notation matter. If you write sloppy labels on tests, you will lose easy marks.

Worked example — reasoned

Q: Are points A, B, C collinear if AB = 3, BC = 5, AC = 8?

  1. What it asks: Do A, B, C lie on the same straight line?
  2. Key fact: If B is between A and C, AB + BC = AC (Segment Addition Postulate).
  3. Check numeric: 3 + 5 = 8 → AB + BC = AC holds.
  4. Conclusion: Yes — the three points are collinear and B is between A and C.
Practice:
  1. If P(0,0), Q(2,0), R(4,0) are points on a plane, are they collinear? Yes — all have y = 0 (same line).
  2. Give an example of three coplanar but not collinear points. Any three points forming a triangle, e.g. (0,0),(1,0),(0,1).
  3. State the postulate that says “through any two points there is exactly one line.” Through any two points there is exactly one line.
Exam tips: Label diagrams clearly. When asked if points are collinear/coplanar always give a reason (e.g. AB+BC=AC, or same line equation). If coordinate points are given, plug into line equation to check.
