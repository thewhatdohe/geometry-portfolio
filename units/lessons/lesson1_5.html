1.5 — Angle Relationships & Proofs

  • Identify complementary, supplementary, adjacent, linear pair, and vertical angles
  • Use algebra to find angle measures
  • Write short two-column proofs for basic theorems

Key relationships: Complementary: sum 90°. Supplementary: sum 180°. Vertical: opposite angles formed by intersecting lines — they are equal. Linear pairs are adjacent and supplementary.

Worked example — algebraic angles

Q: ∠LMN = (4x − 2)°, ∠PQR = (9x + 1)°, and they are complementary. Find x and both angles.

  1. Write equation: (4x − 2) + (9x + 1) = 90.
  2. Simplify: 13x − 1 = 90 ⇒ 13x = 91.
  3. x = 7. Compute angles: ∠LMN = 4(7) − 2 = 26°; ∠PQR = 9(7) + 1 = 64°.
  4. Check: 26 + 64 = 90°, correct.

Proof (two-column style)

StatementReason
1. Two lines intersect.Given.
2. ∠1 and ∠2 form a linear pair.Definition (adjacent and on straight line).
3. ∠1 + ∠2 = 180° and ∠2 + ∠3 = 180°.Linear Pair Postulate.
4. ∠1 + ∠2 = ∠2 + ∠3.Transitive/Equality.
5. ∠1 = ∠3.Subtraction Property (subtract ∠2).
  1. If two angles form a linear pair and one is 110°, find the other. 70° (supplementary → 180 − 110).
  2. Find x if angles (3x + 10) and (2x − 5) are supplementary. 3x+10 + 2x−5 = 180 ⇒ 5x +5 =180 ⇒ x=35 ⇒ angles: 115°, 70°.
  3. Explain why vertical angles are congruent in your own words. Two intersecting lines create opposite angles that share the same rays; using linear pair equalities we subtract and get equality.
For proofs, write each statement and reason clearly. If stuck, start by labeling what is given and then list which postulate or theorem can connect to the goal.
