Distance Formula:

  • Distance between points (x₁, y₁) and (x₂, y₂): √((x₂-x₁)² + (y₂-y₁)²).
Midpoint Formula:
  • Midpoint between points (x₁, y₁) and (x₂, y₂): ((x₁+x₂)/2, (y₁+y₂)/2).
Example:
  • Points A(2,3), B(4,7)
  • Distance = √((4-2)² + (7-3)²) = √(4 + 16) = √20
  • Midpoint = ((2+4)/2, (3+7)/2) = (3,5)
